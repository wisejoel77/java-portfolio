# 🧑‍💼 Employee System

A Java project demonstrating **Inheritance** using an employee domain model.  
Models different types of employees with shared and specialized behaviors.

---

## ✨ Features

- Base `Employee` class with common properties  
- Specialized classes: Full-time, Part-time, Intern  
- Method overriding for role-specific behavior  
- Clean and reusable class hierarchy  

---

## 🧠 Concepts

- Inheritance (base and derived classes)  
- Method overriding  
- Code reusability  
- Polymorphic behavior  

---

## 📁 Files

- `Employee.java` → Base class  
- `FullTimeEmployee.java` → Full-time employee  
- `PartTimeEmployee.java` → Part-time employee  
- `Intern.java` → Intern role  
- `Main.java` → Demo and test cases  

---

## 🎯 Goal

To demonstrate how inheritance:
- Promotes code reuse  
- Reduces duplication  
- Enables flexible class design
