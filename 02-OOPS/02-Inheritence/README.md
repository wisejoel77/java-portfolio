# 🧬 Inheritance Projects

A collection of Java projects demonstrating the concept of **Inheritance** using real-world domain models.

Inheritance allows classes to reuse properties and behavior from existing classes, improving code structure and scalability.

---

## 📚 Projects

### 1️⃣ Employee System  
A role-based employee model demonstrating hierarchy and specialization.  
👉 [View Project](./01-employee-system)

### 2️⃣ Shape System  
A geometric model demonstrating shared structure and overridden behavior.  
👉 [View Project](./02-shape-system)

### 3️⃣ Vehicle System  
A transport model demonstrating base and specialized vehicle types.  
👉 [View Project](./03-vehicle-system)

---

## 🧠 Concepts

- Base and derived classes  
- Method overriding  
- Code reusability  
- Hierarchical class design  
- Polymorphic behavior (basics)  

---

## 🎯 Goal

To demonstrate how inheritance:
- Promotes code reuse  
- Reduces duplication  
- Improves maintainability  
- Enables scalable system design
