# 🚗 Vehicle System

A Java project demonstrating **Inheritance** using a vehicle domain model.  
Models different types of vehicles with shared and specialized properties.

---

## ✨ Features

- Base `Vehicle` class  
- Derived classes: Car, Bike, Truck  
- Overridden methods for vehicle-specific behavior  
- Structured and reusable design  

---

## 🧠 Concepts

- Inheritance and hierarchy design  
- Method overriding  
- Code reuse and extensibility  
- Polymorphism basics  

---

## 📁 Files

- `Vehicle.java` → Base class  
- `Car.java` → Car implementation  
- `Bike.java` → Bike implementation  
- `Truck.java` → Truck implementation  
- `Main.java` → Demo and test cases  

---

## 🎯 Goal

To demonstrate how inheritance:
- Reduces redundancy  
- Improves maintainability  
- Enables scalable systems
