# 🔺 Shape System

A Java project demonstrating **Inheritance** using geometric shapes.  
Defines a base shape and extends it into specific shape implementations.

---

## ✨ Features

- Base `Shape` class  
- Derived classes: Circle, Rectangle, Triangle  
- Area calculation using overridden methods  
- Clean hierarchical design  

---

## 🧠 Concepts

- Inheritance and method overriding  
- Abstract/generalized design  
- Runtime polymorphism  
- Code extensibility  

---

## 📁 Files

- `Shape.java` → Base class  
- `Circle.java` → Circle implementation  
- `Rectangle.java` → Rectangle implementation  
- `Triangle.java` → Triangle implementation  
- `Main.java` → Demo and test cases  

---

## 🎯 Goal

To demonstrate how inheritance:
- Simplifies code structure  
- Enables reusable logic  
- Supports scalable design
