# 🎓 Student Profile System

A Java project demonstrating **Encapsulation** using a student domain model.  
Ensures academic and identity data are validated and securely managed.

---

## ✨ Features

- Immutable roll number (cannot be changed)  
- GPA validation (0–10 range)  
- Basic email validation  
- Safe handling of invalid inputs  
- Utility method for topper detection  

---

## 🧠 Concepts

- Private fields with controlled access  
- Immutable identity design  
- Constructor and setter validation  
- Business logic inside classes  
- Defensive programming  

---

## 📁 Files

- `Student.java` → Core model  
- `Main.java` → Demo and test cases  

---

## 🎯 Goal

To demonstrate how encapsulation:
- Preserves identity integrity  
- Prevents invalid academic data  
- Encourages clean OOP design
