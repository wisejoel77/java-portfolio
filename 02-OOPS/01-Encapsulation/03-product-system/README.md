# 🛒 Product Pricing System

A Java project demonstrating **Encapsulation** using a product pricing model.  
Protects pricing data and enforces discount and validation rules.

---

## ✨ Features

- Price validation (no negative values)  
- Discount validation (0–100%)  
- Safe final price calculation  
- Clean currency formatting  
- Controlled access to product data  

---

## 🧠 Concepts

- Private fields with controlled access  
- Validation through setters  
- Business logic encapsulation  
- Defensive programming  

---

## 📁 Files

- `Product.java` → Core model  
- `Main.java` → Demo and test cases  

---

## 🎯 Goal

To demonstrate how encapsulation:
- Protects business rules  
- Prevents invalid pricing states  
- Improves system reliability
